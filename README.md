## UMKC CS470 - Restaurant Database and View
This project is a learning project for COMP-SCI 470: Introduction to Database Management Systems at the University of Missouri-Kansas City.


### Project Description
The primary goal of this project was to create both a database and a program capable of viewing and querying it. 
The view created for the database was done so in Visual Studio and utilizes Windows Forms. Organized into tabs regarding Restaurants, Employees, and Menus,
the application allows users to search the database for individual fields or a combination of many.
